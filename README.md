# Tree Based Classification Models

This project implements and compares several tree based and ensemble machine learning algorithms for predicting customer churn using Python.

## Project Overview

The customer churn dataset was preprocessed and divided into training and testing sets. Multiple classification models were trained and evaluated to compare their predictive performance.

Model performance was assessed using accuracy, confusion matrices, ROC curves, and Area Under the Curve (AUC).

## Models Implemented

* CART Decision Tree
* Bagging Classifier
* Random Forest Classifier
* Gradient Boosting Classifier

## Evaluation Metrics

* Accuracy
* Confusion Matrix
* Classification Report
* ROC Curve
* Area Under the Curve (AUC)

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* rpy2

## Dataset

Customer churn dataset (`mlc_churn`) from the `modeldata` package.
